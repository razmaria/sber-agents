# Идея: LLM-ассистент в виде Telegram-бота

## Описание

LLM-ассистент должен быть выполнен в виде Telegram-бота.

## Основная задача

Основной задачей бота является вести диалог с пользователем и отвечать на его вопросы с ролью, заданной в системном промпте LLM.

## Роль ассистента

**Финансовый гид** — это ИИ-ассистент, который помогает клиентам банка:
- Решать повседневные финансовые задачи
- Получать персональные рекомендации
- Получать мгновенный доступ к информации о продуктах и услугах
